# Aces Barbershop by Menace — Keswick, Ontario

Premium dark-luxury static site for [Aces Barbershop](https://www.acesbarbershopbymenace.com) in Keswick / Georgina, Ontario.

## Features

- Full-bleed hero using shop photography
- White / gold / black / army green palette with glassmorphism
- Mobile-first layout, smooth scrolling, animated counters
- Meet the team with per-barber **Setmore** booking links
- Shop booking iframe, FAQ, Google Maps, contact page
- SEO meta + JSON-LD (`BarberShop`, `FAQPage`, `ContactPage`)
- Accessibility: skip link, focus styles, ARIA accordion, reduced-motion support
